## Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repo**
   ```sh
   git clone https://github.com/genishprakash/food-recipe-page.git
   ```
2. **Navigate to the project folder**
   ```sh
   cd food-recipe-page
   ```
3. **Open the `index.html` file in your web browser**
